## Introduktion
Ett program behöver kunna göra val. Selektion är kodstruktur som testar om 
uttryck är sant eller falskt.
Ett vanligt sätt att utföra det är ```if``` satsen.
```
if (...) {
    // om sant
} else {
    // annars falskt
}
```
